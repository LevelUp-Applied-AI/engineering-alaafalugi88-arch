# SQL Query Results

## Query 1 – All students
This query returns all students stored in the database with their cohort and score.

## Query 2 – High scorers
This query filters students who achieved high scores (score >= 85). It helps identify top-performing students.

## Query 3 – Average score by cohort
This query calculates the average score for each cohort using GROUP BY.

## Query 4 – Students and their projects
This query joins the students and projects tables to show which student worked on which project.
